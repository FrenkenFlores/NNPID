# Курсовая работа

#### Разработка системы управления аэродинамическими винтами БПЛА в крейсерском режиме

### Введение

В системах управления широко применяются ПИД-регуляторы для формирования управляющего сигнала. Благодаря этому выходная величина более точно и быстрее достигает заданного значения. Чтобы ПИД-регулятор работал оптимально, его необходимо настроить перед запуском устройства управления, так как от заданных настроек зависят характеристики переходного процесса — перерегулирование и быстродействие. Под настройкой понимается подбор коэффициентов пропорциональной, интегральной и дифференциальной составляющих. Чаще всего объект управления ведет себя предсказуемо, поэтому подобранные коэффициенты ПИД-регулятора позволяют эффективно управлять объектом, но в тех случаях, когда на объект управления воздействуют многочисленные факторы, предустановленные коэффициенты теряют свою актуальность в ходе процесса. В подобных непредсказуемых условиях применяются методы адаптивного управления, которые отслеживают параметры, действующие на объект управления, и корректируют коэффициенты ПИД-регулятора для поддержания оптимальных характеристик. 

Один из таких методов предполагает использование искусственных нейронных сетей для подбора коэффициентов на основе набора связанных переменных. Такой подход может быть полезен для проектирования системы управления аэродинамическими винтами, где подобранные коэффициенты ПИД-регулятора могут быстро потерять свою актуальность во время полета из-за смены ветра, изменения плотности воздуха и температуры, разряда аккумулятора и возникновения турбулентности. Неправильно подобранные или неактуальные коэффициенты могут привести к перегреву двигателей, повышенному энергопотреблению и сложностям управления.

В данной работе будет спроектирована адаптивная система управления аэродинамическими винтами в крейсерском режиме на основе нейронных сетей с обратной связью по показаниям всех датчиков. В качестве экспериментальной модели будет собрана установка для имитации работы аэродинамического винта в крейсерском режиме, которая состоит из следующих модулей:
- базовый модуль: герметичный контейнер, стержень, основание;
- модуль управления: двигатель, аэродинамический винт, контроллер полета, микрокомпьютер;
- модуль внешних факторов: насос, электрические клапаны, микроконтроллер, батарея, электрическая схема управления напряжением батареи;
- сенсорный модуль: датчик тока, датчик поворота, датчик
  угловой скорости (энкодер), барометр, акселерометры,
  микрофоны.

Объектом исследования выпускной квалификационной работы является система управления аэродинамическими винтами в крейсерском режиме.

Целью данной работы является применение искусственных нейронных сетей для оптимизации управления аэродинамическими винтами в крейсерском режиме.
